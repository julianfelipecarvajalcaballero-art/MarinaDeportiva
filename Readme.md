# Marina Deportiva
## Descripción del dominio
La aplicación Marina Deportiva permite visualizar información de diferentes embarcaciones disponibles en una marina. Los usuarios pueden consultar datos básicos de cada embarcación, buscar elementos, ver detalles y guardar sus favoritos.
## Semana 1
Se desarrolló una pantalla principal con tarjetas de embarcaciones mostrando imagen, nombre y descripción. Se utilizó ScrollView para visualizar todos los elementos y componentes reutilizables para las tarjetas.
## Semana 2
Se implementó una lista utilizando FlatList y una barra de búsqueda que permite filtrar las embarcaciones en tiempo real. También se agregó un sistema de estilos mediante constantes de tema para mantener una apariencia consistente.
## Semana 3
Se incorporó React Navigation 7 con navegación mediante pestañas y navegación entre pantallas. La aplicación permite acceder a una pantalla de detalle para visualizar información específica de cada embarcación.
## Semana 4
Se implementó Zustand para el manejo de estado global. Los usuarios pueden guardar y eliminar embarcaciones de favoritos, y la aplicación muestra la cantidad de elementos guardados en tiempo real.
## Tecnologías utilizadas
* React Native
* Expo
* TypeScript
* React Navigation 7
* Zustand
## Ejecución del proyecto
```bash
cd semana-1
pnpm install
pnpm start
```
(Reemplazar semana-1 por semana-2, semana-3 o semana-4 según la entrega que se desee ejecutar).
